列表标签用来布局
    分类
        无序列表
            <ul>标签 表示无序列表 <li>标签 列表项
            无序标签各个列表项之间没有顺序级别之分，是并列的
            <ul> 只能嵌套<li>标签，不能放其他标签或文字
            <li>标签相当于一个容器，可以容纳所有元素
        有序列表
            <ol>定义有序列表 <li>定义列表项
            有序标签各个列表项之间按一定顺序排列，并以数字显示
            <ol> 只能嵌套<li>标签，不能放其他标签或文字
            <li>标签相当于一个容器，可以容纳所有元素
        自定义列表
            自定义列表对术语或名词进行解释和描述，
            <dl>标签定义自定义列表 <dt>名词 <dd>解释
