    快速生成HTML结构语法
        1 生成标签   div
        2 生成多个标签   div*5
        3 生成父子级关系标签     ul>li
        4 生成兄弟关系标签      div+p
        5 生成带有类名标签      .类名
        6 生成带有id名标签      #id名
        7 生成有序类名          .类名$*n
        8 生成标签内部内容      标签名{内容}
CSS复合选择器
    后代选择器
        1 概述：又称包含选择器，可以选择父元素中的子元素
        2 写法：外层标签写前边内层标签写后边，中间用空格隔开  元素1 元素2 { CSS样式 } 选择元素1里的元素2
        3 标签发生嵌套时，内层标签为外层标签的后代
        4 元素1可以是儿子孙子，只要是元素1的后代即可
        5 元素1元素2可以是任意标签
    子代选择器
        1 子代选择器只能选择某元素最近一级的子元素（亲儿子）
        2 语法：元素1 > 元素2 { 样式声明 }
        3 父元素与子元素用>隔开
        4 元素1是父级元素2是子级 最终选定子级
        5 元素2必须是元素1亲儿子
